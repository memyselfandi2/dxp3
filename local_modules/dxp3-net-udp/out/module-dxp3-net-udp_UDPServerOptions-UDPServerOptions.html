Class: UDPServerOptions

dxp3-net-udp/UDPServerOptions~UDPServerOptions()

new UDPServerOptions()

Properties:
Name Type Description
address String The address we are listening on (default: 0.0.0.0).
destinations Array | String List of ip addresses we send udp messages to. These can be broadcast, multicast or unicast addresses.
encryptionAlgorithm String The encryption algorithm to use.
encryptionKey String Key to be used to encrypt/decrypt messages.
eventMode module:dxp3-net/UDPServerEventMode One of event, message or raw. This determines if the UDPServer will listen for and emit specific events, or emit the decrypted message, or emit the, potentially encrypted, raw message.
ignoreParentProcess Boolean | String True if we should ignore messages send by our parent process. This includes us.
ignoreOurselves Boolean | String True if we should ignore messages send by ourselves.
mode module:dxp3-net/UDPMode One of directed broadcast, limited broadcast, multicast or unicast. The first three will enable the socket to receive broadcast messages.
multicastAddresses Array | String List of ip addresses we send udp messages to. These are by default also set as destinations unless one specifically opts out. Additionally this will set the mode to multicast.
port Number | String The port we are listening on. If any of the destinations does not have a port, this port will be used instead.
reuseAddr Boolean | String Indicates if we should reuse the address. Setting this to true will allow other processes/instances to run on the same machine while still receiving UDP messages.
