setPort(port)

Set the port of this server. Note that in order to set the port the http reverse proxy must not yet have been started (it's state is HTTPReverseProxyState.INITIALIZED) or must currently be stopped (it's state is HTTPReverseProxyState.STOPPED). You may want to check the state first by calling getState().
Parameters:
Name Type Description
port Number | String
